# ShellCheck (TypeScript Edition)

A TypeScript-based static analysis tool for shell / bash scripts that helps catch errors, bad practices and style issues.

## Table of Contents  
- [Introduction](#introduction)  
- [Features](#features)  

## Introduction  
ShellCheck is designed to allow developers to analyse their shell (bash/sh) scripts using a modern TypeScript codebase.  
It aims to uncover common pitfalls, syntax errors, portability issues and style improvements while supporting integration into editor/CI workflows.
